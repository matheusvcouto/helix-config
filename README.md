
### Configuração no macOS (Ghostty)

Adicione o seguinte trecho no arquivo de configuração do **Ghostty** para que a tecla **Option** funcione como **Alt**, assim como no Alacritty:

```toml
macos-option-as-alt = true  # Trata Option como Alt (equivalente a option_as_alt = "Both" no Alacritty)
```
